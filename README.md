# Nexora Solutions - Modern IT Services Website

A fully responsive, premium SaaS-inspired website for Nexora Solutions, an IT Services & Consulting company. Built with React, TypeScript, Tailwind CSS, and Framer Motion.

## 🌟 Features

### Pages
- **Home** - Hero section, services preview, testimonials, and why choose us
- **Services** - Detailed service offerings with features and benefits
- **About** - Company story, mission, vision, team, and timeline
- **Contact** - Contact form, appointment booking, and location map

### Components
- **Navbar** - Sticky navigation with scroll effects
- **Footer** - Company info, links, and social media
- **Chatbot** - Interactive AI assistant UI (frontend only)
  - Quick actions for common queries
  - Mock responses for appointments, services, and general inquiries
  - Smooth animations and typing indicators

### Design Features
- ✨ Clean, minimal, and professional design
- 🎨 Light theme with soft gradients (blue, cyan accents)
- 📱 Fully responsive (desktop, tablet, mobile)
- 🌊 Animated blob background on homepage
- ✨ Smooth hover effects and transitions
- 🎭 Framer Motion animations
- 🎯 Modern typography and spacing
- 💫 Custom scrollbar styling

## 🚀 Tech Stack

- **React 18** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **React Router** - Navigation
- **Framer Motion** - Animations
- **Lucide React** - Icons

## 📦 Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🤖 Chatbot Features

The chatbot is a frontend-only UI simulation with the following capabilities:
- Answer queries about services
- Book appointments (UI only)
- Reschedule meetings (UI only)
- Provide contact information
- Quick action buttons for common tasks
